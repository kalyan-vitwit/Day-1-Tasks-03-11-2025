# Task - 1 (Explaining different git commands)
- **Clone** : _Copies a remote repository (like from GitHub) to your local computer._
```
git clone <repo_url>
```
- **Pull** : _Fetches and merges the latest changes from the remote repo into your local repo._
```
git pull origin main
```
- **Commit** : _Saves your staged changes to the local repository with a message describing what you changed._
```
git commit -m "Added login feature"
```
- **tag**: _Marks a specific commit as important (often used for version releases)._
```
git tag v1.0
```
